## 🚀 Quick Start

### Option 1: Clone & Run Locally
```bash
git clone https://github.com/taqwakfeedback-collab/najd-studio-website.git
cd najd-studio-website
# Open index.html in your browser
```

### Option 2: Deploy to GitHub Pages
1. Push code to GitHub (this repo)
2. Go to Settings > Pages
3. Select `main` branch as source
4. GitHub will provide your URL

### Option 3: Deploy to Other Hosts
This is a static website (no backend needed), so it works on:
- GitHub Pages (FREE)
- Vercel (FREE)
- Netlify (FREE tier available)
- Any web hosting

## 🌍 Connect Your Domain

To connect `najd-studio.online`:

1. **Get your hosting provider's DNS records**
2. **Go to your domain registrar** (where you bought the domain)
3. **Update DNS records** to point to your hosting provider
4. **Wait 24-48 hours** for DNS to propagate

### Example for GitHub Pages:
Add these A records to your domain DNS:
```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;      /* Main brand color */
    --secondary: #8b5cf6;    /* Secondary accent */
    --accent: #ec4899;       /* Highlight color */
    /* ... more colors ... */
}
```

### Add/Edit Testimonials
Open `index.html` and modify the testimonial cards in the testimonials section with real client reviews.

### Update Pricing
Edit pricing in `script.js` and update HTML pricing cards in `index.html`.

### Change Content
All text is in HTML with bilingual data attributes (`data-en` and `data-ar`).

### Add Videos
Replace the video source in the modal:
```html
<source src="your-video-url.mp4" type="video/mp4">
```
